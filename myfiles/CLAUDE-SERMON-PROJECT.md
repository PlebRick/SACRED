This document provides context about my SACRED Bible study app's taxonomy so you can make intelligent recommendations when helping me write sermons.

## How to Use This Information

When helping me write a sermon, please:

1. **Suggest relevant topics** from my taxonomy that fit the passage/theme
2. **Recommend inline tags** to mark illustrations, applications, key points, quotes, and cross-references
3. **Suggest which sermon series** this sermon might belong to (if applicable)
